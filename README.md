# _angular-hands-on_: How to Angular?

## Setting up
1. Install `ng` tool: 
    ```
    npm install -g @angular/cli
    ```
2. Bootstrap: 
    ```
    ng new my-app
    ```
3. What are you waiting for?
    ```
    ng serve --open
    ```
    or as usual,
    ```
    yarn start
    ```
